# Message Card

封筒が画面下から現れ、タップ操作で開封され、中から手紙が表示されるシンプルな Web ページです。  
URL クエリによってテーマや本文を切り替えられる構成になっています。

---

## 概要

- ページ読み込み時に封筒が下から中央へスライド
- 画面タップで封筒を開封
- フラップが奥に回転し、手紙が出現
- 封筒はフェードアウトし、手紙のみ中央に表示
- テーマ・本文を URL クエリで指定可能

---

## 動作イメージ

- 初期表示  
  - 封筒が画面下から出現  
  - 中央に「画面をタップ」表示
- タップ後  
  - フラップが回転して開封  
  - 手紙がスライド表示  
  - 封筒が画面下へ消える

---

## テーマ一覧

`theme` クエリで以下のテーマを指定できます。

- `simple`（デフォルト）
- `cute`
- `classic`
- `christmas`
- `night`
- `modern`
- `modern-dark`

### 指定例

```text
?theme=cute
?theme=modern-dark
```

## 手紙本文の指定

`letter` クエリに手紙本文を指定します。

- 改行は `\n` を使用
- HTML はそのまま描画されるため注意

### 指定例

```text
?letter=こんにちは。\n\nこれはテストメッセージです。
```

## ファイル構成

- 単一 HTML ファイルのみ
- 外部ライブラリ不使用
- CSS / JavaScript はすべて埋め込み

---

## 使用技術

- HTML5
- CSS3
  - CSS Variables（テーマ切替）
  - `clip-path`
  - 3D Transform（`rotateX`）
- JavaScript（Vanilla）
  - `URLSearchParams`
  - クラス切替によるアニメーション制御

---

## レスポンシブ対応

- スマートフォン対応
- 画面幅 600px 以下で以下を調整
  - 手紙の文字サイズ縮小
  - タップ案内テキスト縮小

---

## 使い方

- HTML ファイルをそのまま配置
- ブラウザで直接開く、または Web サーバーに配置
- URL クエリでテーマ・本文を指定

---

## 利用シーン例

- メッセージカード
- サプライズ演出
- 誕生日・記念日ページ
- 期間限定イベントページ
